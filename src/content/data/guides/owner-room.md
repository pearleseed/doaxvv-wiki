# Owner Room Customization

Unlock furniture and increase closeness with the girls.

## Owner Room Basics

The Owner Room is your personal space where you can interact with the girls and build relationships.

## Closeness System

### What is Closeness?
- Relationship level with each girl
- Unlocks new interactions
- Provides bonus rewards

### Increasing Closeness
1. Daily visits to Owner Room
2. Gift giving
3. Completing girl-specific missions
4. Using girl in matches

## Furniture System

### Furniture Types
- Decorative items
- Interactive furniture
- Seasonal items
- Limited collaboration items

### Unlocking Furniture
- Event rewards
- Shop purchases
- Achievement rewards
- Gacha bonuses

## Room Customization

### Layout Tips
- Place interactive items accessibly
- Create themed arrangements
- Rotate seasonal decorations

### Girl Preferences
- Each girl has furniture preferences
- Matching preferences boosts closeness
- Check girl profiles for hints

## Benefits of High Closeness

1. Unlock special episodes
2. Bonus stats in matches
3. Exclusive dialogue
4. Special gifts from girls

## Daily Room Routine

- Visit each girl's room
- Give daily gifts
- Check for new interactions
- Collect closeness rewards
