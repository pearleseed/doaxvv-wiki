# Daily Routine Checklist

Optimize your daily play session to maximize resource gain in just 15 minutes.

## Quick Daily Checklist

### Must-Do (5 minutes)
- [ ] Claim login bonus
- [ ] Collect mailbox items
- [ ] Complete daily missions
- [ ] Use friend supports

### Should-Do (5 minutes)
- [ ] Check event progress
- [ ] Spend natural stamina
- [ ] Visit Owner Room
- [ ] Send friend gifts

### If Time Permits (5 minutes)
- [ ] Farm event points
- [ ] Upgrade accessories
- [ ] Pull on banners
- [ ] Check shop deals

## Weekly Tasks

### Monday
- Weekly mission reset
- Check new event schedule

### Wednesday
- Mid-week event check
- Adjust farming pace

### Weekend
- Ranking push if needed
- Catch up on missed dailies

## Monthly Tasks

- Review V-Stone budget
- Plan for upcoming banners
- Clean up inventory
- Update friend list

## Efficiency Tips

1. Set a daily alarm
2. Don't let stamina cap
3. Batch similar tasks
4. Skip low-value activities
