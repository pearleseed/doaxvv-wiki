# Trend Ball Usage Guide

Learn how and when to use Trend Balls for maximum event points.

## What are Trend Balls?

Trend Balls are special items that boost your event point earnings. They're essential for efficient event farming.

## Types of Trend Balls

### Standard Trend Ball
- 1.5x point multiplier
- Common drop from events
- Use freely

### Super Trend Ball
- 2x point multiplier
- Rare drop or shop purchase
- Save for important events

### Premium Trend Ball
- 3x point multiplier
- Very rare
- Use only for ranking pushes

## When to Use

### Best Times
- During point bonus periods
- When you have full trend team
- Final hours of event ranking

### Avoid Using
- Early in events (save for later)
- When stamina is low
- On easy difficulty matches

## Maximizing Value

1. Stack with trend swimsuits
2. Use during 2x point campaigns
3. Play highest difficulty
4. Combine with stamina items

## Farming Trend Balls

- Event shop purchases
- Event mission rewards
- Login bonuses during events
- Ranking rewards
