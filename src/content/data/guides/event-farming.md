# Event Farming Optimization

Maximize your rewards from every event with these strategies.

## Event Types

### Festival Events
- Point-based ranking
- Use trend swimsuits for bonus points
- Focus on highest difficulty you can clear

### Gacha Events
- Pull for event-specific swimsuits
- Event swimsuits give point bonuses
- Limited time availability

### Tournament Events
- PvP-style competition
- Team composition matters most
- Rewards based on wins

## Farming Strategies

### Maximizing Points
1. Use all trend swimsuits available
2. Play highest difficulty consistently
3. Use stamina efficiently
4. Don't let stamina cap

### Time Management
- Events typically last 1-2 weeks
- Plan your farming schedule
- Front-load farming if possible

## Reward Priorities

### Must-Get Rewards
- SSR tickets
- V-Stones
- Limited items

### Nice-to-Have
- SR tickets
- Upgrade materials
- Cosmetic items

## Efficiency Tips

1. Calculate points needed for target rewards
2. Estimate daily farming requirements
3. Use natural stamina regeneration
4. Save stamina items for final push
