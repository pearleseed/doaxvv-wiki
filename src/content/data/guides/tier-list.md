# Character Tier List

Current meta ranking for all characters in different roles.

## Tier Definitions

### S Tier
- Best in class
- Excels in multiple areas
- High priority for investment

### A Tier
- Very strong
- Good in most content
- Solid investment

### B Tier
- Average performance
- Situationally useful
- Lower priority

### C Tier
- Below average
- Niche uses only
- Not recommended for new players
