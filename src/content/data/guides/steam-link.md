# Steam Link Setup Guide

Play DOAXVV on your mobile device using Steam Link.

## Requirements

### Hardware
- PC running DOAXVV via Steam
- Mobile device (iOS or Android)
- Stable WiFi connection (5GHz recommended)

### Software
- Steam client on PC
- Steam Link app on mobile
- DOAXVV installed on Steam

## Setup Steps

### PC Setup
1. Open Steam and log in
2. Enable Remote Play in Settings
3. Add DOAXVV to Steam library
4. Launch game once to verify it works

### Mobile Setup
1. Download Steam Link from app store
2. Open app and sign in to Steam
3. Connect to your PC on same network
4. Pair the devices

## Optimization

### Network Settings
- Use 5GHz WiFi when possible
- Reduce distance from router
- Close bandwidth-heavy apps

### Quality Settings
- Start with balanced preset
- Adjust based on connection
- Lower resolution if laggy

## Touch Controls

### Basic Controls
- Tap to click
- Drag to move cursor
- Pinch to zoom (if enabled)

### Advanced
- Configure touch regions
- Use external controller
- Customize button layout

## Troubleshooting

### Connection Issues
- Restart Steam on PC
- Check firewall settings
- Verify same network

### Performance Issues
- Lower streaming quality
- Close background apps
- Check PC performance
