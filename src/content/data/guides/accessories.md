# Accessory Upgrade Guide

Best accessories and how to upgrade them efficiently.

## Accessory Types

### Stat Accessories
- Boost specific stats
- Stack with swimsuit stats
- Essential for optimization

### Event Accessories
- Provide event bonuses
- Limited availability
- High priority during events

### Cosmetic Accessories
- Visual changes only
- No stat impact
- Collection purposes

## Upgrade System

### Materials Needed
- Upgrade stones (by rarity)
- Zack Coins
- Duplicate accessories (for limit break)

### Upgrade Priority
1. SSR accessories first
2. Focus on main team
3. Event accessories during events

## Best Accessories

### Universal Picks
- Accessories that work on any character
- Balanced stat boosts
- Good for beginners

### Specialized Picks
- Character-specific accessories
- Maximum stat potential
- For advanced players

## Farming Locations

### Daily Dungeons
- Best source of upgrade materials
- Rotate by day of week
- Use bonus characters

### Event Shops
- Limited accessories
- Upgrade materials
- Check every event

## Efficiency Tips

1. Don't upgrade R accessories
2. Save materials for SSR
3. Plan upgrades around events
4. Check accessory set bonuses
