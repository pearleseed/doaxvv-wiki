# Guest Point Farming

Efficient ways to farm Guest Points for shop items.

## What are Guest Points?

Guest Points are a social currency earned through friend interactions. They can be exchanged for valuable items in the Guest Point shop.

## Earning Guest Points

### Daily Sources
- Using friend support: 10-50 points per use
- Receiving friend support usage: 10-50 points
- Daily friend gifts: varies

### Maximizing Earnings
1. Add active friends
2. Set attractive support characters
3. Use friend supports daily
4. Send and receive gifts

## Friend Management

### Ideal Friends
- Active daily players
- Strong support characters
- Mutual gift senders

### Removing Inactive Friends
- Check last login date
- Remove 7+ day inactive
- Replace with active players

## Guest Point Shop

### Priority Items
1. SSR tickets (if available)
2. V-Stones
3. Upgrade materials

### Skip These
- Common materials
- Low-value items
- Overpriced cosmetics

## Tips

- Max out friend list (50 friends)
- Check shop reset schedule
- Save points for limited items
