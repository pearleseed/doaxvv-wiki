# Translation Helper

Assist with in-game text translation and localization.

## Features

- **Text Extraction**: Extract text from game files
- **Translation Database**: Access community translations
- **Real-Time Translation**: Translate text on-the-fly
- **Custom Dictionaries**: Add your own translations
- **Export/Import**: Share translation files

## Installation

1. Download Translation Helper from the official website
2. Extract to `C:\Tools\TranslationHelper`
3. Run `TranslationHelper.exe`
4. Select your game directory

## Usage

### Extracting Game Text

1. Click "Extract Text"
2. Select the game files to extract from
3. Wait for extraction to complete
4. View extracted text in the editor

### Translating Text

1. Select text to translate
2. Choose target language
3. Enter translation or use auto-translate
4. Save changes

### Applying Translations

1. Go to "Apply" tab
2. Select translation file
3. Click "Apply to Game"
4. Restart the game to see changes

## Tips

- Always backup game files before applying translations
- Use community translations as a starting point
- Test translations in-game before sharing
- Keep translation files organized by version

## Troubleshooting

### Text not appearing in game

- Verify translation file format is correct
- Check if game version matches
- Ensure file encoding is UTF-8

### Garbled characters

- Check font support for target language
- Verify encoding settings
- Try different font options
