# Save Editor

Edit and backup your game save files safely.

## Features

- **Save File Editing**: Modify game progress, items, and stats
- **Automatic Backup**: Creates backups before any changes
- **Multiple Save Slots**: Manage multiple save files
- **Import/Export**: Share save files with others
- **Validation**: Ensures edited saves remain valid

## Installation

1. Download Save Editor from the official website
2. Extract to `C:\Tools\SaveEditor`
3. Run `SaveEditor.exe` as administrator
4. The tool will automatically detect your save files

## Usage

### Opening a Save File

1. Click "Open Save"
2. Navigate to your save file location
3. Select the save file to edit

### Editing Values

- Navigate through the tree view to find values
- Double-click a value to edit it
- Click "Save" to apply changes
- A backup is automatically created

### Restoring Backups

1. Go to "Backups" tab
2. Select the backup you want to restore
3. Click "Restore"
4. Confirm the restoration

## Tips

- Always create a manual backup before major edits
- Test edited saves immediately after changes
- Don't modify values beyond game limits
- Keep backup files in a safe location

## Troubleshooting

### Save file won't load

- Verify the file is not corrupted
- Check if you have the correct game version
- Try restoring from a backup

### Changes not appearing in game

- Make sure you saved the file
- Verify the save file location is correct
- Restart the game completely
