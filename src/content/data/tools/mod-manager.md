# Mod Manager

Manage and organize your game modifications with ease.

## Features

- **Mod Organization**: Keep all your mods organized in one place
- **One-Click Install**: Install mods with a single click
- **Load Order Management**: Control the order mods are loaded
- **Conflict Detection**: Automatically detect mod conflicts
- **Backup System**: Create backups before installing mods

## Installation

1. Download Mod Manager from the official website
2. Extract to `C:\Tools\ModManager`
3. Run `ModManager.exe`
4. Point the tool to your game installation directory

## Usage

### Adding Mods

1. Click "Add Mod" button
2. Select the mod file (.zip or folder)
3. The mod will be automatically detected and added to your list

### Enabling/Disabling Mods

- Check the box next to a mod to enable it
- Uncheck to disable without removing
- Changes take effect after restarting the game

### Load Order

Drag and drop mods to change their load order. Mods at the top load first.

## Tips

- Always backup your game before installing mods
- Read mod descriptions for compatibility information
- Keep your mods updated to the latest versions
- Disable mods one by one to troubleshoot issues

## Troubleshooting

### Mod not appearing in game

- Verify the mod is enabled
- Check load order conflicts
- Ensure mod is compatible with your game version

### Game crashes after installing mod

- Disable recently installed mods
- Check for mod conflicts in the conflict detection tab
- Verify mod files are not corrupted
