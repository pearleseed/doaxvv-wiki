# Screenshot Tool

A powerful utility for capturing high-quality screenshots of your gameplay moments.

## Features

- **High-Quality Capture**: Capture screenshots in full resolution without compression
- **Hotkey Support**: Quick capture with customizable keyboard shortcuts
- **Auto-Save**: Automatically save screenshots to your designated folder
- **Format Options**: Save in PNG, JPG, or BMP formats
- **Overlay Removal**: Option to hide UI elements before capturing

## Installation

1. Download the Screenshot Tool from the official website
2. Extract the files to `C:\Tools\ScreenshotTool`
3. Run `ScreenshotTool.exe` as administrator
4. Configure your preferred settings

## Usage

### Basic Screenshot

Press `F12` (default hotkey) to capture a screenshot of the current game window.

### Custom Hotkeys

1. Open the Settings menu
2. Navigate to "Hotkeys"
3. Click on the action you want to rebind
4. Press your desired key combination
5. Click "Save"

### Screenshot Settings

- **Format**: Choose between PNG (lossless), JPG (smaller file size), or BMP (uncompressed)
- **Quality**: For JPG format, adjust quality from 1-100
- **Save Location**: Set your preferred folder for screenshots
- **Filename Pattern**: Customize how files are named (supports date/time variables)

## Tips

- Use PNG format for the best quality when sharing online
- Enable "Hide UI" mode for cleaner screenshots
- Set up multiple hotkeys for different capture modes
- Use the built-in image viewer to quickly review your captures

## Troubleshooting

### Screenshots are black or blank

- Make sure you're running the tool as administrator
- Check if your game is running in exclusive fullscreen mode
- Try switching to borderless windowed mode

### Hotkeys not working

- Verify no other application is using the same hotkey
- Check if the tool is running in the system tray
- Restart the application and try again

## System Requirements

- Windows 10 or later
- 100 MB free disk space
- Administrator privileges for game capture
