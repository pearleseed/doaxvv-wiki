# Performance Monitor

Monitor game performance and system resources in real-time.

## Features

- **Real-Time Monitoring**: Track FPS, CPU, GPU, and RAM usage
- **Overlay Display**: Show stats while playing
- **Performance Logging**: Record performance data for analysis
- **Alert System**: Get notified of performance issues
- **Customizable Display**: Choose what metrics to show

## Installation

1. Download Performance Monitor from the official website
2. Extract to your preferred location
3. Run `PerformanceMonitor.exe`
4. Configure overlay settings

## Usage

### Enabling the Overlay

1. Launch the Performance Monitor
2. Click "Enable Overlay"
3. Start your game
4. Press `Ctrl+Shift+P` to toggle overlay visibility

### Customizing Metrics

1. Right-click the overlay
2. Select "Settings"
3. Choose which metrics to display
4. Adjust position and size

### Recording Performance

1. Click "Start Recording" before launching the game
2. Play normally
3. Click "Stop Recording" when done
4. View the performance report

## Available Metrics

- **FPS**: Frames per second
- **Frame Time**: Time to render each frame (ms)
- **CPU Usage**: Overall CPU utilization (%)
- **GPU Usage**: Graphics card utilization (%)
- **RAM Usage**: Memory consumption (MB/GB)
- **VRAM Usage**: Video memory usage (MB/GB)
- **Temperature**: CPU and GPU temperatures (Â°C)

## Tips

- Use the overlay to identify performance bottlenecks
- Record performance during problematic areas
- Compare metrics before and after game updates
- Monitor temperatures to prevent overheating

## Troubleshooting

### Overlay not showing

- Verify the tool is running
- Check if overlay is enabled
- Try running as administrator
- Ensure game is not blocking overlays

### Incorrect readings

- Update to the latest version
- Restart the monitoring tool
- Check for driver updates
